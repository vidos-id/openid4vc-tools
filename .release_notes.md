# Release v0.11.0

## Highlights

- Complete refactor of OID4VP response delivery with local transport control, providing a prepare-and-submit API for authorization responses.

## New Features

- **OID4VP Prepare-and-Submit API**: Split the wallet OID4VP implementation into focused request, response, and submission modules with a new prepare-and-submit flow.
- **Transport Injection**: Callers can now inspect the generated request, rewrite the destination URL for localhost or reverse-proxy setups, or deliver the response in-process during tests.
- **Wallet-CLI Enhancements**: Updated presentation handling to prepare submissions first, submit through the new API, and expose prepared submission details for dry-run and test scenarios.
