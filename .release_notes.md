## Highlights

- Added `credential_offer_uri` support so issuers and wallets can complete OID4VCI exchanges using by-reference credential offers resolved from issuer metadata.
- Introduced an issuer web workspace for managed credential issuance, including dashboard and credential detail flows.
- Aligned issuer web credential lifecycle handling with issuer-backed status lists so status reads and writes now stay consistent across the stack.

## New Features

- Added support for by-reference credential offers in OID4VCI flows.
- Added the issuer web workspace for managed credential issuance.

## Bug Fixes

- Switched issuer web status handling to the issuer-backed status list and limited UI actions to valid lifecycle transitions.
- Added test coverage for issuer CLI credential file output paths.

## Other Changes

- Updated package scripts to call `tsc` directly instead of `bun x tsc`.
- Expanded CLI help documentation and steered wallet usage docs toward `wallet-cli help`.
