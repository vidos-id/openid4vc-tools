# Release v0.10.1

## Highlights

- Fixed CLI packages to publish runnable binaries when installed via npm.

## Bug Fixes

- **CLI bins**: Corrected `bin` entry points in `wallet-cli` and `issuer-cli` to point to `./dist/cli.mjs` instead of `./dist/index.mjs`, ensuring `openid4vc-wallet` and `openid4vc-issuer` commands work after npm install.
- **Build config**: Updated `tsdown.shared.ts` to emit both `index` and `cli` entry points for CLI packages.
