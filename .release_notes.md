## Highlights

- Add a minimal OpenID4VCI issuance flow across `@vidos-id/issuer`, `@vidos-id/wallet`, and `@vidos-id/wallet-cli`.
- Support by-value credential offers, issuer metadata discovery, pre-authorized-code exchange, nonce-based JWT proofs, and direct wallet import for `dc+sd-jwt` credentials.
- Add `wallet-cli receive` plus end-to-end verification coverage and a runnable OID4VCI demo script.

## New Features

- Expose issuer-side OID4VCI helpers for metadata generation, by-value credential offers, and `openid-credential-offer://` serialization.
- Add wallet-side OID4VCI offer parsing, metadata fetching, token exchange, proof creation, credential request, and credential storage APIs.
- Add `bun run demo:oid4vci` for a full mocked end-to-end issuance flow.

## Bug Fixes

- Align nonce retrieval with the OID4VCI `POST` nonce endpoint behavior.
- Align credential requests with the spec `proofs.jwt[]` structure and validate issuer-supported proof algorithms.

## Other Changes

- Archive the completed OpenSpec change for minimal OID4VCI issuance.
- Expand issuer, wallet, and CLI documentation to describe the supported subset and explicit out-of-scope features.
