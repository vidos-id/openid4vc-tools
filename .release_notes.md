# Release v0.10.0

## Highlights

- Renamed the project to `openid4vc-tools` and aligned the published package names under the `@vidos-id/openid4vc-*` scope for a more consistent install and import surface.
- Added a release workflow that bumps workspace versions, publishes packages to npm with the correct dist-tag, attaches CLI artifacts to GitHub Releases, and consumes `.release_notes.md` automatically during release creation.

## New Features

- Added `openid4vc-issuer metadata` to fetch and inspect the issuer metadata document from `/.well-known/openid-credential-issuer`, including endpoints, signing keys, and supported credential configurations.

## Other Changes

- Added repository metadata for individual packages and standardized package builds around `tsdown`, including publishable CLI/library outputs.
- Refreshed the repo and package documentation to match the renamed project, updated CLI commands, and current issuer and wallet workflows.
